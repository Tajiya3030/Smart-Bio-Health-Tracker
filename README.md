# Smart Bio Health Tracker

## 📌 Project Overview
Smart Bio Health Tracker is an IoT-based health monitoring system that measures:

- Body Temperature
- Heart Rate (Pulse)
- SpO2 (Oxygen Level)

The system displays real-time data on LCD and sends health updates via SMS using GSM module.

---

## 🚀 Features
✔ Real-time health monitoring  
✔ LCD display output  
✔ SMS alert system  
✔ Portable and low-cost  
✔ IoT-based healthcare solution  

---

## 🛠 Components Used

- Arduino UNO
- MAX30102 Pulse Oximeter Sensor
- DHT11 Temperature Sensor
- 16x2 I2C LCD Display
- GSM Module (SIM800/900)
- Jumper Wires
- Power Supply

---

## ⚙️ Working

1. Sensors collect temperature, pulse and SpO2 data.
2. Arduino processes the values.
3. Data is displayed on LCD.
4. After 60 seconds, SMS is sent to registered phone number.
5. Cycle repeats continuously.
